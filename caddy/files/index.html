Welcome to the demo site for nginx-quic

This is the home for QUIC+HTTP/3 support in nginx.

The code is developed in a separate "quic" branch available at https://hg.nginx.org/nginx-quic based on nginx mainline. We merge new nginx releases into this branch regularly.

The project code base is under the same BSD license as nginx.

The code is currently at a beta level of quality and should not be used in production.
